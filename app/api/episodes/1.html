1. Install Yeoman and the Angular Generator
npm install -g yo # installs yeoman into global npm modules npm install -g generator-angular # installs angular-generator into global npm modules
2. Generate a new project
yo angular # scaffold out a AngularJS project
3. Start dev server
grunt serve # preview your app (formerly `grunt server`)
4. Run default unit tests
grunt test # test your app
5. Build for deployment
grunt # build the application for deployment
6. Install AngularUI dependency
bower install angular-ui --save # install a dependency for your project from Bower
